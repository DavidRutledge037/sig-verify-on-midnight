# Cryptographic Implementation

## Overview
Core cryptographic implementation for Midnight's signature verification system using secp256k1 and BIP32/44 key derivation.

## Components
- CryptoService: Core cryptographic operations
- MidnightKeyManagement: HD wallet functionality
